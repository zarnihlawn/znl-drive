# Settings

Open **Settings** from your profile menu in the top bar.

## Appearance

Tune **theme**, **font**, and **text size** so the app matches your environment and readability preferences. Changes apply to the client session and stored preferences as implemented in `AppSettingsDialog`.

## Language

The app uses **Paraglide** for locales. Pick a language in settings so in-app strings and localized routes (including **`/onboarding/docs`**) resolve consistently.

## Search

The main navbar includes a **Search** field as part of the shell. Behavior may evolve; settings that relate to search will live alongside other preferences in the same dialog.

## Profile

Use **Profile** (also in the menu) for account-related options, including developer features when enabled.
