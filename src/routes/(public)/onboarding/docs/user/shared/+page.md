# Shared with you

The **Shared** view lists files and folders that other users have shared with your account.

## Permissions

In the current product, shared access is oriented around **read** usage in the UI: you can browse and download according to what the owner granted. You cannot create new folders or upload into another user’s share from this view.

## Navigation

Breadcrumbs distinguish **Home → Shared** and, when you open a shared folder, include the folder name. Use the sidebar to jump back to **Home** or **Trash** at any time.
